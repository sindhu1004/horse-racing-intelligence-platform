# Horse Racing Intelligence Platform

Production-grade machine learning platform for horse race outcome prediction using temporal feature engineering, XGBoost, and FastAPI.

## Overview

An end-to-end machine learning system for predicting horse racing outcomes using historical race data.

## Planned Features

- Data ingestion
- Exploratory Data Analysis
- Feature engineering
- XGBoost & LightGBM
- Probability calibration
- SHAP explainability
- Drift detection
- Automated retraining
- FastAPI deployment

## Tech Stack

- Python
- Pandas
- NumPy
- Scikit-learn
- XGBoost
- LightGBM
- PostgreSQL
- FastAPI
- Docker
